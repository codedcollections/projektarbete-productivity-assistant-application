# 🪄 Productivity App

React productivity app powered by Vite. Users can manage todos, habits, and events, with support for user authentication.
[Live Demo](https://projektarbete-productivity-assistan.vercel.app/login)

## 🚀 Quick start

1. Clone the repo:

   ```bash
   git clone https://github.com/codedcollections/projektarbete-productivity-assistant-application.git
   cd projektarbete-productivity-assistant-application
   ```

2. install dependencies

npm install

3. dev server

npm run dev

4. Open the app in your browser:

Visit `http://localhost:5173`

## Available commands

- `npm run dev` — start the Vite development server
- `npm run build` — build the production app
- `npm run preview` — preview the production build locally
- `npm run lint` — run ESLint checks

## Features

- User registration and login
- Personal dashboard overview
- Todo management
- Habit tracking
- Event planner
- User-specific data stored with localStorage and sessionStorage
- API calls for random quote on login
